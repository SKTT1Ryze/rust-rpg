# **Rust RPG**

**A RPG Game Written with Rust.**  

## Dependencies
piston = "0.52.0"  
piston2d-graphics = "0.37.0"  
pistoncore-glutin_window = "0.66.0"  
piston2d-opengl_graphics = "0.74.0"  
log = "0.4.11"  


## Run
```Shell
cd game
cargo run
```
## Control
+ Up, Down, Left and Right
+ Type 1~9 to choose which game object to control
+ Much to do
